== Description ==

Wtyczka WP-To-Blox.pl służy do automatycznej republikacji wpisów z bloga Wordpress na blogu na platformie blogowej Blox.pl. 

Wtyczka jest samoobsługowa i republikacja następuje od razu po publikacji na blogu macierzystym. Oprócz tego wtyczka obsługuje obrazki, linki oraz tag [youtube], którego podanie wraz z kodem w ten sposób: [youtube]kodFilmu[/youtube] powoduje zamieszczenie na Blox.pl podanego filmu (polecam używać z pluginem Video-Embedder dla WP), który taki tak właśnie stosuje.

Rozważ subskrypcję bloga Polishwords aby być na bieżąco z projektem.

ENGLISH

Plugin supports [youtube] tag known from video-embedder plugin.

Subscribe  to Polishwords blog to be up-to-date with new releases.

== Installation ==

1. Wrzuć pliki do foldera '/wp-content/plugins/wp-to-blox'
2. Aktywuj plugin w zakładce 'Plugin' w Twoim blogu Wordpress
3. Ustaw chmod na 77, aby móc edytować wp-to-blox.php
4. Wejdź w edytor w zakładce wtyczki w panelu administracyjnym bloga wordpress, następnie w odpowiednim miejscu podaj hasło i login do Twojego konta na blox.pl

ENGLISH

1. Upload files to the `/wp-content/plugins/wp-to-blox` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set chmod 777 privilages for wp-to-blox.php file
4. Set login and password inside wp-to-blox.php to your blox.pl account login and password

2. Czy wtyczka obsługuje filmy z YouTube?

TAK. Wtyczka nie tylko obsługuje filmy ale dopasowuje je do szerokości blogów na Blox.pl.
